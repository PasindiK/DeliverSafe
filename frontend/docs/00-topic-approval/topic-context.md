# Step 0 — Topic Selection and Approval

## Proposed Domain
IoT-enabled food delivery thermal bag monitoring dashboard for operational decision support.

## Problem Context
Food delivery operators need near real-time visibility of bag conditions (temperature, humidity, tilt, leaks, lid status) to reduce spoilage risk, customer complaints, and safety issues.

## Datasets
- **Primary data:** Sensor streams from delivery bags (temperature, humidity, tilt, leak, lid open, signal quality, route metadata).
- **Secondary backup data (compulsory):** Public weather/environmental data to support contingency analytics when sensor feeds become incomplete.

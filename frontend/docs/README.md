# IT4031 Assignment Documentation Pack

This folder structure maps directly to the assignment steps and rubric.

## Folder Mapping
- `00-topic-approval` → Step 0 evidence (approved topic, dataset justification)
- `01-empathize` → Stakeholders, interviews, questionnaires, personas, empathy maps
- `02-define` → Problem statements, user stories, HMW, journeys, task flows
- `03-ideate` → IA, navigation, sketches, low-fi wireframes, chart rationale
- `04-prototype` → Hi-fi prototype screenshots, Figma links, rationale
- `05-test` → Usability test plan, participant profile, findings, metrics
- `06-iterate` → Before/after comparisons and design refinements
- `07-handoff` → Implementation-ready handoff specifications
- `08-ai-disclosure` → Mandatory AI tools usage disclosure

## Team Working Rule
- Each member should add initials to sections they authored.
- Use real evidence only (no fabricated personas, data, or usability results).
- Keep all raw artifacts (forms, transcripts, screenshots) referenced from each section.

# SmartDelivery Dashboard (React)

Responsive visual analytics dashboard starter for food delivery monitoring:
**IoT-enabled food delivery bag monitoring with contingency weather backup data**.

## Tech Stack
- React + TypeScript + Vite
- Recharts for interactive visualizations

## Quick Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Project Structure
```text
docs/                   Assignment evidence templates (Steps 0–7 + AI disclosure)
data/                   Primary and secondary dataset placeholders
src/
  components/
    cards/              KPI components
    charts/             Trend and anomaly charts
    filters/            Dashboard filter controls
    lists/              Alert table
  features/dashboard/   Page-level dashboard composition
  services/             Mock data and analytics/selectors
  styles/               Dashboard styling
  types/                Shared TypeScript models
```
